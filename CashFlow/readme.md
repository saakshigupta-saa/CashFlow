# 💰 CashFlow Tracker

A modern and responsive Cash Flow Tracker Dashboard built using HTML, CSS, and JavaScript. The application helps users track income, expenses, remaining balance, and visualize spending with interactive charts.

---

## 🚀 Features

- Responsive Dashboard
- Add Income & Expenses
- Summary Cards
- Budget Progress Tracker
- Expense Analytics (Chart.js)
- Recent Transactions
- Search Transactions
- PDF Report Download
- Dark & Light Mode
- Local Storage Support
- Help & Support Page
- Mobile Friendly Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- jsPDF
- Font Awesome
