# 💰 Cash Flow Dashboard

A modern and responsive **Cash Flow Dashboard** built using **HTML, CSS, and JavaScript**. This project helps users manage their personal finances by tracking income, expenses, remaining balance, and spending patterns through an interactive dashboard.

## 🚀 Features

- 📊 Interactive Dashboard
- 💵 Income & Expense Tracking
- 📈 Expense Analytics with Chart.js
- 📉 Budget Progress Tracker
- 📝 Recent Transactions History
- 🔍 Search Transactions
- 📄 Download PDF Reports
- 🌙 Dark & ☀️ Light Mode
- 💾 Local Storage Support
- 🆘 Help & Support Page
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- jsPDF
- Font Awesome
